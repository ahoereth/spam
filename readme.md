# Cognitive Science Study Planning #
[cogsci.uos.de/~SPAM](https://cogsci.uos.de/~SPAM)

SPAM is a study planning machine for the major Cognitive Science at the University of Osnabrueck.

## Build ##

Requirements: Node >= 0.10

	$ npm install
	$ bower install
	$ grunt build
